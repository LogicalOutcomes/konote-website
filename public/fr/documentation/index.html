Notes sur la documentation

Documentation évolutive

La documentation est maintenue en parallèle avec le code dans le dépôt GitHub. Si vous trouvez des erreurs ou des lacunes, vous pouvez soumettre des corrections via une demande de tirage ou ouvrir un ticket.

Compatibilité des versions

La documentation reflète la branche principale actuelle. Si vous utilisez une version plus ancienne de KoNote, certaines fonctionnalités ou instructions peuvent différer. Consultez l'historique des commits si vous avez besoin de la documentation d'une version spécifique.
