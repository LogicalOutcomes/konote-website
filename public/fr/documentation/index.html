Notes sur la documentation

<div class="notice notice-info">
  <p class="notice-title">Documentation évolutive</p>
  <p>
    La documentation est maintenue en parallèle avec le code dans le dépôt GitHub.
    Si vous trouvez des erreurs ou des lacunes, vous pouvez soumettre des corrections via une demande de tirage
    ou ouvrir un ticket.
  </p>
</div>

<div class="notice notice-warning">
  <p class="notice-title">Compatibilité des versions</p>
  <p>
    La documentation reflète la branche principale actuelle. Si vous utilisez
    une version plus ancienne de KoNote, certaines fonctionnalités ou instructions peuvent différer.
    Consultez l'historique des commits si vous avez besoin de la documentation d'une version spécifique.
  </p>
</div>
