## What KoNote Is

KoNote is an open-source, self-hosted participant outcome management platform for Canadian social service nonprofits. It tracks participant progress, measures outcomes, and generates funder reports. Built with Django/PostgreSQL, it deploys via Docker on Railway, Azure, OVHcloud, or self-hosted infrastructure.

## What KoNote Is Not

KoNote is not a SaaS product — there is no hosted version. It is not a billing system, clinical EMR, or CRM. It does not have a public API, native mobile app, or video conferencing. There is no vendor support line — it is community-supported open source software with optional professional services.

## Privacy and Hosting

Organizations host KoNote on their own infrastructure and control their own data. All personal information is encrypted at rest with per-agency keys. The system is designed for Canadian privacy requirements (PIPEDA/PHIPA). The hosting organization is responsible for server security, backups, and updates.

## Bilingual Support

KoNote has a full English/French bilingual interface following canada.ca conventions. All terminology can be customized in both languages. The website is available in both English and French.
